Share files via a distributed Interplanetary File System (IPFS) on your device
Hardware Required: None (Pi only))
> More...
The IPFS application shares files from the sync directory on your device with the IPFS network
If you have deployed the IPFS application:
  - Files placed in the sync folder will be made available on the public IPFS network.
  - Added files not instanteneously available on the IPFS network, it takes time to populate files information across IPFS nodes. Files are usually available within 10-15 minutes.
  - Please consider that files, once added to IPFS, are likely to be there forever even if you delete them from the sync folder (similar to a version control system).
  - Note IPFS consumes network bandwidth (10-15GB per day) even if your files are not being downloaded as it does block exchange with the BitSwap protocol. IPFS is still in alpha, so capping bandwidth consumption has not been implemented yet.
Once the registration is complete, your IPFS node will be shown on Blue Horizon map and accessible on the IPFS network.
